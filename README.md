# Событийно-ориентированный симулятор СМО

Минимальная, чистая реализация для курсовой работы по теме:
**"Механизмы эффективного распараллеливания в задачах 
событийно-ориентированного имитационного моделирования"**

## Особенности реализации

1. **Минималистичный дизайн** - только необходимые компоненты
2. **Чистый код** - понятные структуры, хорошие комментарии
3. **Основные распределения**:
   - Exponential (поток Пуассона)
   - Uniform (равномерное)
   - Deterministic (детерминированное)
   - Erlang (распределение Эрланга)
4. **Одна дисциплина очереди** - FIFO (достаточно для демонстрации)
5. **Многопоточный сервер** - поддержка нескольких ядер
6. **Подробная статистика** - все основные метрики СМО

## Сборка и запуск

```bash
# Сборка
make

# Запуск
make run

# Очистка
make clean
